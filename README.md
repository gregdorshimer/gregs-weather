View NWS forecasts for U.S. cities.

React-google-maps is used with use-places-autocomplete and react-select's AsyncSelect component to fetch locations and geocode them.

The NWS weather API is used to get forecasts.

OpenStreetMap and OpenWeatherMap are used to get the street and radar layers for the leaflet map, which is made using react-leaflet.

NWS API:
https://www.weather.gov/documentation/services-web-api

OpenStreetMap:
https://www.openstreetmap.org/about/api/

OpenWeatherMap:
https://openweathermap.org/api

Find forecast office codes, or use gridpointURL below:
https://en.wikipedia.org/wiki/List_of_National_Weather_Service_Weather_Forecast_Offices
